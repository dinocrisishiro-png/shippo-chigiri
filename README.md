# ググガガ しっぽちぎり

ペンギン「ググガガ」のしっぽをちぎる 60 秒スコアアタックゲーム。
`index.html`（Vanilla JS / Canvas、外部ライブラリなし）と `assets/` のスプライト PNG で動きます。

## 遊び方

1. ググガガに**そーっと**近づく（ポインタを速く動かして近づくと「！」で気づかれて逃走される）
2. しっぽをタップ / クリックで掴む（マウス・タッチ両対応）
3. ググガガが逃げる方向と**逆**にドラッグして張力ゲージをためる
4. ゲージが満タンになると「ちぎり！」でスコアゲット（連続でちぎるとコンボ倍率）
5. しっぽは数秒で再生。60 秒間でハイスコアを目指そう

ベストスコアは `localStorage` に保存されます。

## ググガガのステートマシン

```
のんびり ──(ポインタが速く接近)──▶ 警戒(！) ──▶ 逃走
   ▲                                            │
   └────────(ほとぼりが冷める)───────────────────┘
のんびり/逃走中 ──(しっぽを掴む)──▶ 捕獲 ──(ちぎり！/すっぽ抜け)──▶ 逃走
```

## 実行

`index.html` をブラウザで開くだけ。ビルド不要。

## スプライト描画

- `assets/gugugaga_idle.png`（正面向き・全ステート共通）と `assets/tail.png` を `drawImage` で描画
- 左向き移動時は左右ミラー、警戒ステートは「！」オーバーレイで表現
- しっぽの付け根は体スプライトのお尻位置（`SPRITE_DEFS.idle.butt`）、掴み判定はしっぽ画像の先端（`SPRITE_DEFS.tail.grab`）の相対座標アンカーで定義
- `assets/tail.png` は本体画像の黒い生地部分を切り出してティアドロップ型に加工したもの
- 画像のロードに失敗した場合は従来の図形描画に自動フォールバック
